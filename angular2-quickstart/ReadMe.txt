How To Run
==========

1- Clone the repository
2- npm install
3- npm start

You can follow https://angular.io/docs/ts/latest/quickstart.html

Files Structure
===============

1- package.json 
lists packages the QuickStart app depends on and defines some useful scripts.

2- tsconfig.json 
It is the TypeScript compiler configuration file.

3- typings.json 
It identifies TypeScript definition files.

4- systemjs.config.js, 
In is the SystemJS configuration file.
